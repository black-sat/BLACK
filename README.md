BLACK (short for Bounded Lᴛʟ sAtisfiability ChecKer) is a temporal reasoning framwork for Linear Temporal Logic and related logics.

This is the **development** branch, which is undergoing a major overhaul of the whole system.

Please refer to stable releases if you want to use BLACK.
